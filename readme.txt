== Description ==

Widget Logic - New Version Widget Logic allow you to control on which pages widgets appear using WP's conditional tags. It also adds a 'widget_content' filter.

== Installation ==

1. Download plugin zip file to your computer and extract in a folder
1. Upload `simple_widget_logic` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
